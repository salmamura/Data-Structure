			=== PETUNJUK PEMBUATAN FILE HEADER & DRIVER PADA CODEBLOCKS ===

1. Klik New File pada toolbar, lalu pilih Project, atau
   Pada Menu Bar pilih : File -> New -> Project

2. Lalu akan muncul pilihan untuk memilih wizard type (pada bagian kanan), pilih Files

3. Pilih C/C++ header

4. Klik Next, lalu tentukan nama dan lokasi untuk menyimpan file header (tidak perlu mengisi/edit Header guard word)

5. Klik Finish, header file akan langsung terbuka

Untuk pembuatan Driver (Main) File sama seperti pembuatan file.c biasanya (New File -> Empty File)
